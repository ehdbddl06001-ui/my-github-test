# 오답노트 시스템

틀린 문항을 **과목별로 분리 저장**하여 반복 복습과 약점 분석이 가능하도록 만든 구조입니다.

## 구조

```
오답노트/
├── README.md        ← (이 파일) 운영 방법
├── _양식.md          ← 오답 기록 표준 양식 (복사해서 사용)
├── 01_의학총론.md
├── 02_순환기.md
├── ... (과목별 파일)
└── 16_의료관계법규.md
```

## 기록 방법

### ① 자동 기록 (권장) — 대화형 퀴즈
[`../quiz/quiz.py`](../quiz/quiz.py)로 문항을 풀다가 **오답이 나오면 이 폴더의 과목 파일에 자동으로 기록**됩니다.
기록일·내가 고른 답·정답·핵심 정리·감별 포인트가 채워지고 상단 요약 카운트(🔴)가 증가합니다.
```bash
cd ../quiz && python3 quiz.py --subject 순환기
```
자동 기록된 항목에서 "틀린 이유(지식 공백)"과 "복습 예정일"만 직접 채우면 됩니다.

### ② 수동 기록
1. 틀린/헷갈린 문항이 나오면 해당 과목 파일을 엽니다.
2. [`_양식.md`](_양식.md)의 표를 복사하여 파일 하단에 붙여넣고 채웁니다.
3. "내가 고른 답 / 정답", "틀린 이유(지식 공백)", "핵심 정리"를 반드시 채웁니다.
4. **복습 예정일**을 정해 간격 반복(spaced repetition)으로 다시 봅니다.

> 복습 모드: `python3 ../quiz/quiz.py --review` 를 실행하면 오답노트에 쌓인 문항만 다시 출제합니다.

## 과목 분리의 이점

- **약점 과목 정량화**: 과목 파일별 오답 개수를 세면 취약 영역이 드러납니다.
- **집중 복습**: 시험 직전 취약 과목 파일만 골라 회독할 수 있습니다.
- **과락 관리**: 특히 `16_의료관계법규.md`는 과락(전 과목 40% 미만 시 불합격) 위험이 큰 과목이므로 별도로 집중 관리합니다.
