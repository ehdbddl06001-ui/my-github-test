# 오답노트 기록 양식

아래 표를 복사하여 해당 **과목별 오답노트 파일** 하단에 붙여넣고 채우세요.

```markdown
### [오답 #N] 문항 제목/키워드

| 항목 | 내용 |
|------|------|
| 기록일 | 2026-00-00 |
| 과목 | (예: 순환기) |
| 세부 주제 | (예: 급성 심근경색 초기 처치) |
| 출처 문항 | (예: 문항/02_순환기.md 문항 1) |
| 내가 고른 답 | ② |
| 정답 | ④ |
| 틀린 이유(지식 공백) | (왜 틀렸는지 — 개념 오해 / 계산 실수 / 함정 등) |
| 핵심 정리 | (반드시 기억할 한두 줄) |
| 감별 포인트 | (헷갈린 보기와의 임상적 차이) |
| 관련 개념 연결 | (연관 개념·타 과목 연결) |
| 복습 상태 | 🔴 |
| 복습 예정일 | 2026-00-00 |
```

## 작성 예시

### [오답 #1] STEMI 재관류 시간 창

| 항목 | 내용 |
|------|------|
| 기록일 | 2026-07-01 |
| 과목 | 순환기 |
| 세부 주제 | 급성 ST분절 상승 심근경색(STEMI) 재관류 전략 |
| 출처 문항 | 문항/02_순환기.md 문항 1 |
| 내가 고른 답 | ② 혈전용해제 우선 투여 |
| 정답 | ④ 즉시 일차적 관상동맥중재술(primary PCI) |
| 틀린 이유(지식 공백) | PCI 가능 병원 도착~풍선확장 90분 이내면 PCI 우선인 것을 놓침 |
| 핵심 정리 | STEMI: PCI 가능하면 door-to-balloon ≤90분 목표로 primary PCI 우선 |
| 감별 포인트 | 혈전용해제는 PCI 불가·이송 120분 초과 시 고려 |
| 관련 개념 연결 | 불안정 협심증/NSTEMI는 위험도 평가 후 침습 전략 |
| 복습 상태 | 🟡 |
| 복습 예정일 | 2026-07-08 |
