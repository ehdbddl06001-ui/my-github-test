# KMLE 문항 제작 세트

한국 의사 국가시험(KMLE) 스타일의 문항과 임상·기초 의학을 함께 다루는 케이스형 해설을 정리한 저장소입니다.

## 구성 원칙

- **문항 스타일**: 실제 KMLE와 유사하게 임상 증례(vignette) → 질문 → 5지선다(①~⑤) 형식으로 구성했습니다.
- **케이스형 해설**: 각 해설은 정답의 **핵심 개념 + 기초의학적 기전 + 임상적 적용**을 함께 다룹니다.
- **출처 명시**: 해설마다 근거가 되는 대표 교과서/가이드라인을 표기했습니다.
- **보기별 임상적 차이(감별)**: 모든 오답 보기가 "왜 답이 아닌지"를 임상적으로 감별할 수 있게 설명했습니다.
- **과목별 2문항**: 아래 16개 과목에서 각 2문항씩, 총 32문항을 제작했습니다.

## 오답노트 사용법

- 틀린 문항은 해당 과목의 [오답노트](오답노트/) 파일에 [기록 양식](오답노트/_양식.md)에 맞춰 누적합니다.
- 과목별로 파일이 분리되어 있어, 약점 과목을 빠르게 파악하고 반복 복습할 수 있습니다.
- 자세한 운영 방법은 [오답노트/README.md](오답노트/README.md)를 참고하세요.

## 주의

- 본 문항과 해설은 **학습용**이며, 실제 진료 판단은 최신 가이드라인과 임상 상황에 따라야 합니다.
- 약제 용량·검사 수치는 학습 시점의 대표값을 사용했으므로, 최신 개정판을 함께 확인하세요.
