# MIT-BIH Inter-Patient SVEB 정확도 향상 연구 — 실험 종합 정리

**목표**: MIT-BIH 부정맥 DB, inter-patient(DS1→DS2) 설정에서 SVEB(S)의
`S_PR-AUC`를 올린다. 원본 baseline(`cnn_base_psa2`)의 벽은 S≈0.6.

**핵심 결론(요약)**:
1. 백본·손실·구조·환자적응·앙상블·라우팅 등 **"정보 재조합" 계열은 전부 노이즈
   바닥(±0.05)에서 정체.** 이 데이터로 짜낼 수 있는 건 사실상 다 짜냄.
2. **P파 벡터-에너지 특징("정보 추가")만이 유일하게 label-free 판별 정보를 실제로
   더함**(선형 로지스틱 inter-patient S 증분 +0.06~0.08, 누출 검증 통과). 단
   단일-비트 CNN이 raw 파형에서 이미 상당 부분을 흡수해 15-seed에서 순증은 미미.
3. **병목은 방법이 아니라 데이터**: DS1 환자 22명, DS1 SVEB 944비트. 모든 실험이
   이 지점을 가리킴. 실질적 상향은 외부 데이터(환자 수·S 수 확대)가 유력.

---

## 0. 설정과 저울(측정 프로토콜)

- **데이터**: `mamba_data.npz` — beat[N,2,300], ref[N,2,300], feats[N,26], y(0=N/1=S/2=V), pid, t. N=99,871.
- **클래스 분포**: N 90,082(90.2%) / **S 2,781(2.8%)** / V 7,008(7.0%). DS1 S=944, DS2 S=1,837.
- **분할**: de Chazal DS1(22명)→학습, DS2(22명)→테스트. inter-patient(환자 겹침 없음).
- **주 지표**: `S_PR-AUC`(희소 클래스라 accuracy·ROC-AUC는 낙관적 → PR-AUC가 정직).
- **핵심 방법론적 발견 — 앙상블 저울**: 개별 seed의 DS2 S_PR-AUC는 ±0.15~0.19로
  극심하게 요동. **seed별 확률을 평균한 앙상블**이 분산을 흡수(개별 대비 +0.06~0.08)
  하며, 이후 모든 비교의 공식 저울로 사용.
- **노이즈 바닥**: "동일 설정"의 서로 다른 run이 앙상블 S에서 ±0.05~0.07 차이
  (GPU 비결정성 + 극단적 seed 민감성 + 희소 S). 이보다 작은 개선은 재현 불가.

## 1. 재조합(reshuffling) 계열 — 전부 노이즈 바닥

측정은 모두 label-free ref + gss 분할 + 앙상블 기준(달리 명시 없으면).

| Step | 방법 | 결과(앙상블 S) | 판정 |
|---|---|---|---|
| 1 | Focal loss | 0.57→0.57 | 무익/악화 |
| 1 | SupCon + 균형샘플러 | 0.48 | **악화**(oversampling이 inter-patient 해침) |
| 1.5 | StratifiedGroupKFold 분할 | 0.55(균일 0.65 leaky→0.55) | 무익, V도 하락 |
| 3 | **FiLM 학습 환자적응** | 0.53→0.43 | **악화** — DS1 22명 과적합 |
| 3b | 바운드 FiLM(γ=tanh) | 0.49 | 붕괴는 막았으나 base와 동급 |
| 3c | 자기예측 정상 ref(base) | 0.50→0.50 | 미미 |
| 4 | **AdaBN(파라미터-free 적응)** | 0.475→0.516 | **분산 반감**(0.14→0.08), 평균 소폭↑, V −0.04 |
| 5 | 깨끗한 ref + AdaBN α스윕 | 0.43~0.46 | 스택 안 됨(노이즈) |
| 6 | temporal(이웃 beat 형태) | S 0.48→0.30, V↑ | **S 독, V 약**(S 신호는 이웃형태 아님) |
| 7 | P파 attention + 클래스 분기 | 0.58→0.52 | 순증 없음 |
| grand | 57모델 균일 앙상블 | 0.567 | — |
| grand2 | 계열 라우팅(V←temporal, S←그외) | 0.595 | +0.028(합법 label-free) |
| grand2 | confidence 라우팅 | ≤0.591 | 계열라우팅 못 넘음 |
| 8 | 학습 스태킹(환자CV) | (실행시 판정) | 라우터 일반화 테스트 |

**교훈**:
- **학습된 환자적응(FiLM)은 22명으로 과적합** → 새 환자에 악화. inspection으로 경계가
  환자별로 형성됨을 확인했으나 일반화는 별개였음.
- **파라미터-free 적응(AdaBN)은 일반화**하여 분산을 줄임(평균은 소폭). "환자마다 다른
  함수"는 맞되, 22명으로 **학습**하면 안 되고 **환자 통계로 정렬**해야 한다는 대비.
- 앙상블·라우팅·스태킹의 이득은 대부분 **분산 흡수**이지 새 정보가 아님.

### 1-1. 하드니스 분석(놓친 S의 정체)
DS2 진짜 S 1,837비트를 "57모델 중 몇 개가 맞히나"로 분류:
- **아무도 못 맞힘 3.4%**(정보 없음, 조합 불가) / 1~10%만 7.6% / 10~50% 32.8% / 50%+ 56.2%.
- per-beat 오라클 recall 0.966은 **57모델 max의 통계적 착시**(confidence 라우팅이
  계열라우팅을 못 넘은 게 증거). 실사용 라우팅으로 그 착시를 못 먹음.

## 2. 정보 추가(P파 벡터-에너지) — 유일하게 벽을 넘은 축

**아이디어**: SVEB 판별 정보 = 조기성(RR, 이미 feats에 존재) + **P파 형태/축 이상**.
avg-pool CNN이 작은 P파를 QRS에 묻어버림. 그래서 **환자 정상 기준의 P파 특징**을 추가.

**8개 특징(step9, 환자 정상 ref 기준 = 개인 함수)**: VM=√(l1²+l2²) 리드독립 에너지,
atan2 벡터방향(이소성 축), P/QRS/T 에너지분해, residual=숨은 P(P-on-T) 등.

### 판별력(학습 없이, DS2 단변량 ROC-AUC)
- P잔차(이상P) **0.802**, P비율(정상대비) **0.775**, 전체형태편차 **0.766**,
  전체VM비 **0.738**, P피크위치 0.663. → **여러 특징이 진짜 S 신호를 담음.**

### 증분(선형 로지스틱, DS1학습→DS2, inter-patient)
| 정상 기준 | 26+P파8 S | 증분 |
|---|---|---|
| (a) 정상템플릿 ref [누출] | 0.472 | +0.204 |
| (b) 전체 median ref [LF] | 0.324 | **+0.057** |
| (c) 자기예측 정상 ref [LF·깨끗] | 0.349 | **+0.081** |
- **누출(정상 라벨) 걷어내도 label-free 증분 +0.06~0.08 생존** → P파 정보는 실재.
- V도 +0.27(0.545→0.814)로 크게 오르나, CNN V는 이미 0.97 포화라 전이 미미.
- 자기예측 정상 ref가 가장 강함 → "환자 정상 기준을 깨끗이 잡을수록" 판별력↑
  (환자별 함수 테마와 연결).

### CNN 검증(step9d, label-free 자기예측 ref)
- 5 seed: base26 0.433 → pwave 0.519(+0.086) — 노이즈바닥 초과처럼 보였으나
- **15 seed: base26 0.544 → pwave 0.501(−0.043)** — **+0.086은 seed 1001 운빨,
  15seed에서 소멸.** V는 그대로.
- **크루드8+분리8 합본(step11, 15 seed): base26 0.494 → combined 0.406(−0.088)** —
  16개는 오히려 CNN을 **해침**(크루드는 흡수, 분리8은 노이즈 추가로 특징MLP 교란).
- **해석**: P파 정보는 로지스틱엔 크게 기여하나, **CNN은 raw 파형에서 이미 흡수** →
  요약특징으로 또 줘도 순증 없고, 많이 주면 오히려 해로움. **P파 축은 CNN용으로 소진.**

### 벡터 분리(denoise, step10)
- 환자 정상 P축 기준 coh(정렬=진짜P)/perp(이탈=노이즈·이소성) 분리 + 저역통과.
- 증분: 분리8 +0.030 < 크루드8 +0.057. 합본 +0.066(크루드 대비 +0.009).
- **더 정교한 분리가 더 많은 정보를 주진 못함**(1D 투영 손실, 노이즈 낀 축추정).

## 3. 종합 결론

1. **재조합(손실·분할·적응·앙상블·라우팅·스태킹)은 이 단일 데이터에서 노이즈 바닥에
   수렴.** 특히 "학습된 환자적응 FiLM은 22명 과적합, 파라미터-free AdaBN은 일반화"의
   대비는 그 자체로 의미 있는 음성 결과.
2. **정보 추가(P파 벡터-에너지 특징)는 유일하게 label-free 판별 정보를 실제로 더함**
   (로지스틱 inter-patient S +0.06~0.08, 누출 검증 통과, 물리적으로 타당). 그러나
   **단일-비트 CNN이 이를 흡수**하여 최종 성능 순증은 미미. 더 정교한 분리(denoise)도
   크루드를 못 넘음.
3. **병목은 방법이 아니라 데이터**: DS1 22명·S 944비트. 하드니스 분석상 진짜 정보
   부재는 3.4%로 작으나, 재현 가능한 상향에는 **환자 수·S 수의 확대(외부 데이터:
   PTB-XL/INCART 등)** 가 유력. 이는 학습 환자적응의 과적합 해소와 희소성·분산 해소를
   동시에 달성.

## 4. 기여(논문화 가능한 요소)
- 앙상블을 공식 저울로 한 **엄밀한 inter-patient 프로토콜 + 노이즈 바닥 정량화**.
- **학습 환자적응(FiLM) vs 파라미터-free 적응(AdaBN)** 의 일반화 대비.
- **P파 벡터-에너지 특징이 label-free로 실재 판별 정보를 더한다**는 검증(+누출 통제).
- **"CNN이 raw에서 흡수" 진단**: 로지스틱엔 크게 기여하나 CNN 순증 없음 →
  정보의 소재(feature)와 모델 용량(capacity)을 분리한 세련된 음성 결과.

## 5. 향후(외부 데이터 경로)
- PTB-XL/Chapman/INCART 등을 wfdb로 수집 → AAMI(N/S/V) 매핑 → 리드·샘플레이트 정합 →
  DS1과 결합해 **환자 수 대폭 확대**. inter-patient 순수성 위해 DS2(테스트)는 불포함.
- 환자 수가 커지면 (a) 학습 환자적응(FiLM/hypernetwork)이 비로소 일반화 가능,
  (b) 희소 S 분산 감소, (c) P파 특징의 신호가 CNN 흡수분을 초과할 여지.

## 6. 선행연구 비교 주의 (intra vs inter-patient)
- Ait Bourkha et al.(IJACSA 2025), "Optimized WSN and CNN": MIT-BIH 15클래스
  **98.5% accuracy**. 그러나 **비트 무작위 80/20(클래스 층화) 분할 = intra-patient**
  (동일 환자 비트가 train·test 공존) + **SMOTE** + 전체 accuracy. → 과대평가된
  쉬운 세팅으로, 우리의 inter-patient S_PR-AUC와 **비교 불가**. 이 논문은 inter-patient를
  측정하지 않음. (첫 GPT PDF가 경고한 "동일환자 train/test 공존 과대평가"의 실례.)
- **교훈**: "데이터 충분"은 **intra-patient엔 참, inter-patient엔 거짓**. 우리 결론 유지.
- **빌릴 것**: WST(Wavelet Scattering) 특징 추출법은 유효 → 우리 inter-patient·label-free·
  증분 검증 세팅에서 테스트(step12). 우리 방법론이 차별점.
